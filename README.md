## Capstone Design Project

### Visualization Simulator for Cosmetic Manufacturing Vessels Using Manufacturing Data
*Smart Factory Capstone Design 1 · Team 12*

Developed a Population Balance Model (PBM)-based simulator with multiple quality indicators to predict and judge scale-up emulsification quality in cosmetic emulsion manufacturing vessels (CEVSim).
- Responsible for results analysis and project management
- Analyzed droplet size distribution data to evaluate emulsion quality and predict manufacturing outcomes
- Project outcomes resulted in a pending patent application and a registered software work (see *Research Outcomes* below)
